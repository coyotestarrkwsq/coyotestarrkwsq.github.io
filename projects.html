PUDDLESTORE

2019

PuddleStore is a fault-tolerant, strongly consistent distributed file system. It uses Tapestry to store data blocks and Raft consensus algorithm to commit changes. Implemented using Go.

GREAT DINNER GUEST

2019

Great Dinner Guest is a web application that connects people to experts in a certain field at a designated restaurant where the guest (expert) can share his/her knowledge to the client. Implemented using React, Express, MySQL and AWS.

NEURAL STYLE TRANSFER FOR VIDEOS

2018

A program that applies art filter to videos, implemented using tensorflow. Coherent stylized video frames are produced by training with optical flow incorporated in the loss function.
